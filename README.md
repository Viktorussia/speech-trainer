# 🧠 Speech Trainer (Aphasia)

Интерактивный онлайн-тренажёр для восстановления речи у пациентов с афазией.

👉 Демо: https://speech-trainer-tau.vercel.app/

---

## 🚀 Функционал

- 📂 Категории (посуда, одежда, техника, животные и др.)
- 🔊 Озвучка слов (кастомная аудио + fallback синтез речи)
- 🧩 3 уровня сложности:
  - Слушай и выбирай
  - Соотнеси слово и картинку
  - Собери слово из букв
- 🎯 Прогресс выполнения
- 📱 Адаптация под мобильные устройства

---

## 🛠 Технологии

- React
- Vite
- JavaScript (ES6)
- CSS (адаптивный UI)

---

## 🎯 Назначение

Проект разработан как логопедический инструмент для:
- восстановления номинативной функции речи
- работы с пациентами с афазией
- самостоятельных тренировок
